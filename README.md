# Project Cadastral Cauayan (CaCau)
An open-source technical project that aims to fix discrepancies within the official boundaries of the barangays of Cauayan City published by the Cauayan City Planning and Development Coordinators' Office.

**DISCLAIMER: This project does not intend to put any negativity towards the Cauayan City Planning and Development Coordinators' Office.**

## Institutional Call to Action
The project is still in the status of proposal since February 27 after failure to contact the Isabela State University and Northeastern College.
